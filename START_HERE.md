# RoleForge — démarrage rapide

## Lancer l'application sur Mac

```bash
cd /chemin/vers/RoleForge
bash scripts/run_mac.sh
```

## Envoyer les changements sur GitHub

```bash
git status
git add .
git commit -m "Décrit clairement la modification"
git push
```

## Tester les règles de base

```bash
python -m pytest
```

## Principe de travail

On avance par petites versions stables. Chaque version doit se lancer avant de passer à la suivante.
