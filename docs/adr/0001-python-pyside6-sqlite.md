# ADR-0001 — Stack V1

## Décision
La V1 utilise Python, PySide6 et SQLite.

## Raisons
- Cohérent avec les compétences actuelles.
- Rapidité de prototypage.
- Application desktop locale.
- Aucune dépendance serveur obligatoire.

## Conséquence
L’architecture doit rester modulaire pour permettre plus tard une version web ou un moteur séparé.
