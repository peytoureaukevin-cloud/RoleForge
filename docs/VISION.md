# RoleForge — Vision

RoleForge est une application desktop de jeu de rôle narratif solo, pensée comme un produit premium et non comme un simple chatbot.

La V1 se concentre sur trois fondations :

1. un Atelier du héros riche et agréable ;
2. une Salle d’aventure jouable ;
3. une IA branchable, mais jamais centrale dans l’identité du produit.

Le héros est le cœur du produit. L’utilisateur doit aimer le construire, le modifier, le retrouver et le voir évoluer.
